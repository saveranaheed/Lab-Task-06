# Lab-Task-06
Using  nested For Loop
```cpp
#include <iostream>
using namespace std;
int main() {
  for (int i=1;i<=7;i++)
  {
    for(int j=1;j<=i;j++)
    {
      cout<<"*";
  }
  cout<<endl;
}
}
```
